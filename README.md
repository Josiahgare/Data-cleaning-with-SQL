# Data-cleaning-with-SQL

This project is demonstrate my ability to clean and prepare high quality bikes and cycling accessories data using SQL. It contains SQL automation to aid future data cleaning and preparation of similar data.

**Tool used:** Microsoft SQL Server

**Data source:** KPMG Virtual Internship with Forage

---

## Project Overview

This project involves;
+  **Data Cleaning:** Removing nulls, trimming data, checking invalid data, checking data consistency

+  **Data Transformation:** Transforming data to suit analysis and meet best practices

+  **Data Preparation:** Creating new coumns from existing columns for clearer analytics

---

### Dataset

Three dataset were provided;
- Customer address
- Customer demographics
- Transactions

---

### Process Involved
- CSV files were loaded into SQL server
- Missing data, inconsistent data, inaccurate data were resolved
- Data exploration
- Feature engineering
- Creation of new table **CustomerList** from existing and clean data
- Exporting data as CSV file for future use
